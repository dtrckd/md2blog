
| Design style name | Spirit / vibe | Basic characteristics |
|---|---|---|
| **Neo‑brutalism** | Bold, outspoken, “anti‑polish” but intentional | Thick borders, flat fills, high contrast, big type, blocky buttons, playful geometry, visible grid/structure |
| **Brutalism (web)** | Raw, rebellious, utilitarian | Minimal decoration, harsh typography, stark colors, “unstyled” HTML feel, function-first layouts |
| **Swiss / International** | Clean, rational, editorial | Strong grids, left-aligned type, lots of whitespace, sans-serif fonts, clear hierarchy, minimal ornament |
| **Bauhaus‑inspired** | Functional, modernist, geometric | Primary colors, simple shapes, strong composition, minimal yet expressive geometry |
| **Memphis** | Playful, quirky, 80s-postmodern | Bright colors, random geometric shapes, patterns, asymmetry, fun mixed elements |
| **Flat design** | Simple, modern, efficient | No heavy shadows/textures, solid colors, clean icons, clarity over realism |
| **Material Design** | Structured, friendly, systematic | Consistent spacing, elevation/shadows, cards, motion rules, component-driven UI |
| **Skeuomorphism** | Familiar, tactile, “real-world” | Realistic textures, depth, materials (leather/metal), lifelike controls, detailed shadows |
| **Neumorphism (Soft UI)** | Soft, calm, “pillow-like” | Subtle inner/outer shadows, low contrast, same-color surfaces, minimal outlines (accessibility challenges) |
| **Glassmorphism** | Sleek, airy, modern | Frosted blur panels, translucency, soft highlights, layered depth, gradient backdrops |
| **Claymorphism** | Cute, 3D, toy-like | Puffy elements, soft 3D shadows, rounded shapes, saturated colors, “clay” surfaces |
| **Minimalism** | Quiet, premium, focused | Limited palette, lots of whitespace, simple typography, few components, strong restraint |
| **Maximalism** | Expressive, energetic, loud | Layering, bold colors, mixed type, dense visuals, decorative elements, intentional chaos |
| **Editorial / Magazine** | Sophisticated, content-forward | Strong typography, grids/columns, big headlines, image-led sections, refined spacing |
| **Corporate / Enterprise UI** | Trustworthy, safe, practical | Neutral palettes, predictable patterns, clear forms/tables, conservative typography, consistency |
| **Y2K** | Late-90s/early-2000s glossy futurism | Chrome/metal looks, shiny gradients, bubble type, sparkles, translucent blobs, tech motifs |
| **Vaporwave** | Nostalgic, dreamy, ironic | Neon gradients, pink/purple/cyan, retro 3D, VHS/glitch, classical imagery, surreal layouts |
| **Cyberpunk / Tech‑noir** | Intense, gritty, futuristic | Neon accents on dark, high contrast, HUD elements, glitch effects, dense UI overlays |
| **Retro / Vintage** | Warm, nostalgic, handcrafted | Muted palettes, classic serif type, grain/noise, badges, traditional layouts, print-like feel |
| **Hand‑drawn / Doodle** | Human, casual, friendly | Sketchy lines, imperfect shapes, scribbles, marker fonts, playful illustrations |
| **Playful / Cartoon UI** | Fun, approachable, youthful | Rounded forms, bright colors, big icons, bouncy motion, friendly typography |
| **Monochrome / Duotone** | Bold, graphic, focused | One color or two-tone scheme, strong contrast, simple shapes, consistent mood |
| **Gradient / Aurora** | Modern, soft, atmospheric | Smooth multi-color gradients, glow, subtle noise, abstract blobs, light depth cues |
| **Dark mode / High-contrast** | Dramatic, modern, accessibility-driven | Dark surfaces, bright accents, careful contrast, reduced glare, emphasis on readability |
| **Terminal / Console UI** | Hacker/engineering vibe, utilitarian | Monospace type, text-first layouts, green/amber on dark, minimal graphics, dense info |
| **Dashboard / Data-dense UI** | Analytical, efficient, serious | Tables, charts, filters, compact spacing, consistent components, hierarchy for scanning |
