## Windows

| Action | Hotkeys |
| ------ | ------- |
| Create new split window | Ctrl + Shift + Enter |
| Previous or next window | Ctrl + Shift + ] or [ |
| Change layout | Ctrl + Shift + L |
| Close window | Ctrl + w |

## Tabs

| Action | Hotkeys |
| ------ | ------- |
| New tab | Ctrl + t |
| Close tab | Ctrl + q |
| Previous or next tab | Ctrl + Shift + Right or Left |
| Next layout | Ctrl + Shift + L |
